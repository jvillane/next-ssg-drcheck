## Getting Started

Ejecutar el servidor de desarrollo en [http://localhost:3000](http://localhost:3000):

```bash
npm run dev
```

Empaquetar la aplicación:

```bash
npm run build
```

Ejecutar la aplicación en modo producción:

```bash
npm start
```
